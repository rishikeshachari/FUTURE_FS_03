# 💍 Rishikesh Achari Jewellery – Official Website

A complete, production-ready frontend website for a traditional gold jewellery shop in Anantapur, Andhra Pradesh.

![Website Preview](https://via.placeholder.com/800x400?text=Jewellery+Website+Preview)

---

## 📌 Project Overview

This website helps a local jewellery business establish an online presence, attract customers, and display live gold rates. Built with pure HTML, CSS, and JavaScript – no frameworks, no backend required.

**Live Demo:** [View Website](https://rishikeshachari.github.io/FUTURE_FS_03/)  
**Business:** Rishikesh Achari Jewellery, Anantapur  
**Established:** 2014

---

## ✨ Features

### Core Features
- ✅ **Live Gold Rates** – Auto-updating 24K, 22K, 18K gold prices
- ✅ **Product Gallery** – 39+ jewellery products across 7 categories
- ✅ **Smart Filtering** – Filter by Chains, Earrings, Bangles, Rings, Necklaces, Pendants
- ✅ **Contact Form** – Validated form with FormSubmit integration (no backend needed)
- ✅ **Google Maps** – Embedded location of the shop in Anantapur
- ✅ **WhatsApp & Call Buttons** – Floating action buttons for instant contact
- ✅ **Responsive Design** – Works perfectly on mobile, tablet, and desktop

### Premium UI/UX
- ✨ Loading screen with logo animation
- ✨ Smooth hover effects on product cards
- ✨ Gold shimmer text animations
- ✨ Category badges on products
- ✨ Back-to-top button
- ✨ Floating social contact buttons

### Performance
- 🚀 Lazy loading images
- 🚀 Optimized animations
- 🚀 Mobile-first design
- 🚀 Fast load times

## 📡 API Integration

The website fetches live gold rates from:
- **Metals-API** – Live gold prices in INR
- **GoldPriceIndia** – Fallback API

*Note: For production, replace demo API keys with your own free keys.*
